# Optical-spiral

Artical of website:

在这个项目中，我们将会从解析法进行光强上的光学螺旋（光学弹簧）的生成进行探究。另外我们将尝试对光学弹簧中的螺旋尾巴问题进行讨论。这个问题主要发生在使用单个相位板，其中令两个贝塞尔光相位直接叠加，并使用大面积光源（平面光或者高斯光）照射，产生光学弹簧的过程中发生，本文将会从数学推导出发进行分析。

In this project, we will explore the generation of optical helices (optical springs) on light intensity using analytical methods. Additionally, we will attempt to discuss the issue of helical tails in optical springs. This problem mainly occurs when using a single phase plate, where two Bessel light phases are directly superimposed and illuminated with a large-area light source (plane light or Gaussian light) to produce an optical spring. This article will analyze it from a mathematical derivation perspective.
